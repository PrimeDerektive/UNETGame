thank u for purchasing "warbot"

if u want to play the demscene u will need to unzip the psettingswarbot folder and overwrite your default ones.
or make sure the player controller is not colliding with the feet physics layer.


controls:
wasd to move
space to jump
shift to run
left mouse to attack or shoot when aiming
rightmouse to aim
